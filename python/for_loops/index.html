  1. Create a program which loops 10 times

    • Each time, ask the user for a number
    • Check if this number is a valid number
    • IF it is, add it to a running total
    • ELSE output “not a valid number”
    • When the loop is complete, output the total

This program is going to contain two parts.

First is the function which checks whether a number is valid (an int or float).

Where I found the function

def isNumeric(s):
   s = s.strip()
   try:
      s = float(s)
      return True
   except:
      return False

The remaining code carries out the rest of the program.

total = 0

for i in range(10):
  num = input("enter a number")
  if isNumeric(num) == True:
    total += int(num)
  else:
    print("that was not a valid number")
    
print(total)
