Creating a program which asks the user to type in a sport.

  1. Assign this value to a variable called sport.
  2. If the sport is football, output “The beautiful game”
  3. If the sport is ice hockey, output “The violent game”
  4. If the sport is rugby, output “The game of egg chasers”
  5. If the sport is basketball, output “Netball for men”
  6. Else output “what sport do you like then”
sport = input("Type in a sport: ")

if sport == "football":
    print("The beautiful game")
elif sport == "ice hockey":
    print("The violent game")
elif sport == "rugby":
    print("The game of egg chasers")
elif sport == "basketball":
    print("Netball for men")
else:
    print("What sport do you like then")

*Create a program which asks the user to input two integers (store these values in two variables called x and y) and a mathematical operator (+, -, / or ) and store this in a variable called operator.

  1. IF the operator is a + symbol, output the value of x + y
  2. IF the operator is a - symbol, output the value of x - y
  3. IF the operator is a * symbol, output the value of x * y
  4. ELSE output the value of x / y
print("enter x")
x = int(input())

print("enter y")
y = int(input())

print("enter an operator, + - * or / only")
operator = input()

if operator == "+":
    print(x, "+", y, "=", x+y)
elif operator == "-":
    print(x, "+", y, "=", x-y)
elif operator == "*":
    print(x, "*", y, "=", x*y)
elif operator == "/":
    print(x, "/", y, "=", x/y)

Create a program which asks the user to enter an amount in £ - store this value in a variable called amount.

  1. Ask the user for a currency to exchange the amount to ($, Yen, Euros or NZ $) – store this in a variables called exchange.
  2. IF the exchange variable is $ then output the amount multiplied by 1.5
  3. IF the exchange variable is Yen then output the amount multiplied by 20
  4. IF the exchange variable is Euros then output the amount multiplied by 1.3
  5. IF the exchange variable is NZ $ then output the amount multiplied by 2
  6. ELSE output “we don’t exchange this currency”
print("ENTER AMOUNT IN POUNDS")
poundValue = int(input())

print("WHICH CURRENCY DO YOU WISH TO CONVERT TO")
currency = input("ENTER Dollar, Yen, Euro, or NZ")

if currency == "Dollar":
    op = 1.5
elif currency == "Yen":
    op = 20
elif currency == "Euro":
    op = 1.3
elif currency == "NZ":
    op = 2

    print("We don't trade this currency")


print("YOUR NEW BALANCE IS: ", poundValue * op)
