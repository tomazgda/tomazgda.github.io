Validation

March 5, 2021

The following code will check the length of the string ‘password’. If it is less than six, it will print “Invalid”.

    password = “password123”

    IF password.length < 6 THEN
    PRINT “Invalid”
    ELSE
    PRINT “Valid”
    END IF

The following code will check ‘age’ against the numbers 10 and 19. If it is greater than 10 and less than 19, it will print “senior school pupil”. If not it will print “too old or too young”.

    age = 15

    IF age > 10 AND age < 19 THEN
    PRINT “senior school pupil”
    ELSE
    PRINT “too old or too young”
    END IF

The following code will check if age is an integer.

age = 15

    IF age is integer THEN
    PRINT “this is an acceptable value”
    ELSE
    PRINT “this isn’t a number”
    END IF

The following code will check if the variable ‘name’ contains and data.

    INPUT name

    IF name NOT NULL THEN
    PRINT “hello ” + name
    ELSE
    PRINT “you didn’t enter a name”
    END IF

The following code will check if ‘formGroup’ is in the format ‘LNLLL’.

    INPUT formGroup

    IF formGroup IN FORMAT LNLLL THEN
    PRINT “you are in ” + formGroup
    ELSE
    PRINT “this form group is not recognised”
    END IF
